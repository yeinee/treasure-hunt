== README

1. 서비스 배경
  Around 어플리케이션의 <달콤창고>는 작은 기부 문화의 한 형태로, 서울 시내 몇몇 지하철 역의 물품 보관함에 작은 상품을 넣어두고 다른 이들과 SNS로 비밀번호를 공유하는 서프라이즈 선물 릴레이. 달콤창고의 위치를 검색할 수 있는 달콤지도까지 등장할 정도로 많은 호응을 얻음. 이에 모티프를 얻어 간식거리가 아닌 처치곤란이지만 누군가에게는 유용할 수 있는 작은 물건들을 나누자는 의도로 본 서비스를 기획함. 
  
2. 서비스 대상 및 물품
  * 대상 : 1차적으로 연세대학교 학생을 대상으로 함. 
  * 물품 : 우리가 실제로 쓰지는 않지만 버리기에는 아까운 물건 / 각종 티켓 / 나누고 싶은 물품 등
